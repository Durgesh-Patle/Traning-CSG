Map Tag in HTML.

  • The map tag is used to define an image map. An image map is an image with clickable areas.
  • The required name attribute of the map element is associated with the usemap attribute of the image to create a relationship between the image and the map.
  • The map element contains a number of area elements, which define the clickable areas in the image map.
